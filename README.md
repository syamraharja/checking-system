# Windows System Checker

A simple Node.js-based system checker to retrieve battery status, CPU information, memory usage, operating system details, and disk layout on a Windows machine. Built with the powerful [systeminformation](https://www.npmjs.com/package/systeminformation) library.

## Features

✅ Battery status (charging, percentage, time remaining)  
✅ CPU manufacturer, brand, cores, and speed  
✅ Memory (total, used, free)  
✅ Operating system platform and release  
✅ Disk layout and capacity  

## Requirements

- [Node.js](https://nodejs.org/) (v14 or higher recommended)
- npm

## Installation

```bash
npm install systeminformation
